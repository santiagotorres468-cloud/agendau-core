// Método estático de utilidad
public static function calcularPromedio(array $data): float
{
    $campos = ['p1_claridad','p2_puntualidad','p3_dominio_tema','p4_utilidad','p5_ambiente'];
    $sum = array_sum(array_map(fn($c) => (float)($data[$c] ?? 0), $campos));
    return round($sum / count($campos), 2);
}

Flujo de importación (HorarioController::importar)

1. Validar extensión del archivo (xlsx, xls, csv).
2. Leer encabezados con PhpSpreadsheet → detectar formato (ancho vs. largo).
3. Instanciar HorariosImportWide o HorariosImport según el formato.
4. Excel::import($import, $archivo)
   → Cada fila pasa por model() / collection():
       a. Detectar columnas con alias normalizados.
       b. Parsear horas (fracción decimal Excel → HH:MM:SS).
       c. Validar límite horario institucional (22:00 semana / 17:00 sábado).
       d. Validar cruce de horarios (query DB).
       e. Buscar o crear User (profesor) por correo → nombre.
       f. Insertar HorarioAsesoria → $insertados++.
5. Retornar sesión: import_popup, import_insertados,
   import_fuera_rango, import_conflictos, import_otros.
6. Vista: modal popup muestra contadores y listas de fallos.

Middleware ForzarCambioPassword

// app/Http/Middleware/ForzarCambioPassword.php
public function handle(Request $request, Closure $next): Response
{
    if (auth()->check() && auth()->user()->cambiar_password) {
        return redirect()->route('password.forzar.mostrar');
    }
    return $next($request);
}

Rate limiting de login

// routes/web.php
RateLimiter::for('login', function (Request $request) {
    return Limit::perMinute(5)->by($request->email . $request->ip());

8Módulo de importación Excel

Dos formatos soportados

ClaseFormatoColumnas clave
HorariosImportLargo — una fila por horarioCURSO, DOCENTE, CORREO, DIA, INICIO, FIN, MODALIDAD…
HorariosImportWideAncho — una fila por curso, columnas por díaCURSOS, PROFESOR, LUNES, MARTES, MIÉRCOLES, JUEVES, VIERNES (celda: "8am a 10am")

Detección automática de columnas

HorariosImport usa un mapa de aliases normalizado. Cada columna canónica (ej: curso) puede reconocerse con variantes escritas diferente (ej: materia, asignatura, NOMBRE CURSO). La normalización elimina tildes y convierte a minúsculas antes de comparar.

Conversión de hora Excel

Excel puede guardar horas como fracciones decimales del día (e.g., 0.333… = 08:00). Ambos importers detectan este caso:

if (is_float($value) && $value >= 0 && $value < 1) {
    $mins = (int) round($value * 1440); // 1440 = minutos en un día
    return sprintf('%02d:%02d', intdiv($mins, 60), $mins % 60);
}

Validaciones aplicadas (por fila)

  1. Columnas obligatorias presentes.
  2. Curso y docente no vacíos.
  3. Hora de inicio y fin parseables.
  4. Hora fin ≤ límite institucional (22:00 L–V, 17:00 S).
  5. Sin cruce de horario del mismo docente en el mismo día.
ℹ️
Creación automática de docenteSi el correo del Excel no coincide con ningún usuario existente, el sistema crea automáticamente la cuenta con contraseña temporal Pascual2026 y cambiar_password=true.

9Autenticación

Docentes y administradores — Laravel Jetstream + Sanctum

Jetstream provee las vistas de login, registro y recuperación de contraseña. Sanctum gestiona las sesiones con tokens de API. El guard activo es auth:sanctum.

Estudiantes — Google OAuth (Socialite)

12Consideraciones de seguridad

MecanismoImplementación
CSRFBlade @csrf en todos los formularios POST/PUT/DELETE. Jetstream incluye verificación automática.
Rate limitingLimit::perMinute(5) por email + IP en el endpoint de login.
Autorización por rolComprobaciones explícitas auth()->user()->rol !== 'admin' + abort(403).
Separación de sesionesEstudiantes en sesión PHP; docentes/admins en Sanctum. No hay mezcla de contextos.
Contraseña temporal forzadaMiddleware ForzarCambioPassword intercepta toda ruta privada si cambiar_password=true.
Desactivación de cuentasCampo activo=false + validación en LoginController para rechazar login sin borrar datos.
Validación de archivosExcel solo acepta mimes:xlsx,xls,csv con máximo 5 MB.
SQL InjectionTodas las consultas usan Eloquent ORM o Query Builder con parámetros enlazados.
XSSBlade escapa automáticamente con {{ }}. No se usa {!! !!} con datos de usuario.

13Mantenimiento

Comandos útiles

php artisan config:clear
php artisan route:clear
php artisan view:clear

# Volver a compilar caché para producción
php artisan optimize

# Ver logs de errores
tail -f storage/logs/laravel.log

# Modo de mantenimiento
php artisan down --message="Actualización en progreso" --retry=60
php artisan up

Copias de seguridad recomendadas

  • Dump diario de la base de datos: mysqldump agendau > backup_$(date +%F).sql
  • Backup del directorio storage/ para conservar logs y archivos subidos.
  • Commit del código fuente a repositorio Git tras cada actualización.

Actualizar dependencias

# Verificar dependencias desactualizadas
composer outdated

# Actualizar con restricciones semánticas del composer.json
composer update --with-dependencies

# Siempre ejecutar después de actualizar
php artisan migrate
php artisan optimize
🛑
Base de datos en producciónNunca ejecutes php artisan migrate:fresh en producción. Usa solo migrate o migraciones reversibles para no perder datos.
